## Alternate Launch Options

### Guided Skillmap Session
Use this for most classrooms and workshops. Learners follow the MakeCode Arcade pathway in order and build one concept at a time.

### Full Project Import
Use this when you want learners to inspect or remix the full repository after the guided build.

1. Open MakeCode Arcade.
2. Choose the GitHub import option.
3. Paste the repository URL.
4. Open the project in the simulator.

## Optional Facilitation Supports

- use the session test track when you want a fast compare-and-explain checkpoint
- prepare a short printed or projected reminder of the team roles if career connections are a core goal
- keep one backup demo device ready for fast recovery in live events

## Follow-Up Ideas

- invite teams to remix art, scoring, or setup choices after the core path is complete
- share the repository link with educators who want to inspect the project structure more deeply
- use the project wiki for contributor-facing workflow notes and implementation details
