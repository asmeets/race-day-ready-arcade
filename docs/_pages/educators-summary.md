Use this section for the quick context adults need before they open the experience with learners.

## What Driven by STEM Is

Driven by STEM is a Microsoft MakeCode Arcade project built for the F1 x Mercedes x Microsoft STEM and career exploration activation. It uses an F1-inspired racing game to make engineering decisions visible through play, testing, and remixing.

The experience is designed for middle school learners, but the delivery model also works well for mixed-age events where some participants are new to coding and some are ready to experiment immediately.

## Why It Works in a Classroom or Event

- the game gives immediate feedback, so learners can see what a code change does right away
- the project focuses on tradeoffs instead of a single right answer, which supports discussion and iteration
- the MakeCode Arcade format keeps controls simple and lowers setup friction
- the same activity can support a short demo, a guided lesson, or a remix challenge

## What Adults Should Expect

Learners will spend most of their time testing choices, comparing outcomes, and talking through why one setup performs differently from another. The role of the facilitator is to guide attention toward cause and effect, not to lecture through every mechanic.

## Best Fit Use Cases

- classroom STEM lessons
- coding club sessions
- museum or activation floor demos
- family learning events
- engineering career exploration workshops
