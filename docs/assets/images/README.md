# Image Assets for Driven by STEM Site

This directory should contain the following images for the Jekyll site:

## Homepage (Splash Layout)

- **header-bg.jpg** (1920x600px recommended)
  - Hero background image for the homepage
  - Should feature F1/racing theme with Mercedes branding
  - Will be overlaid with teal color (#00D2BE) at 50% opacity

## Feature Row Thumbnails

- **students-thumb.jpg** (400x300px)
  - Represents student experience
  - Could show: student coding, MakeCode blocks, game simulator

- **educators-thumb.jpg** (400x300px)
  - Represents educator resources
  - Could show: classroom setting, lesson materials, facilitation

- **about-thumb.jpg** (400x300px)
  - Represents the partnership
  - Could show: F1 car, Mercedes team, Microsoft logo combination

## Optional Assets

- **logo.png** - Site logo/icon (transparent PNG, 200x200px)
- **og-image.jpg** - Social media preview image (1200x630px)

## Notes

- All images should be optimized for web (compressed)
- Use progressive JPEGs for faster loading
- Consider retina/2x versions for high-DPI displays
- Ensure images follow F1, Mercedes, and Microsoft brand guidelines
- Images should be licensed for educational use

## Placeholders

For now, the site will function without these images. The theme will gracefully handle missing images.

To add an image:
1. Add the file to this directory
2. Reference it in markdown files as `/driven-by-stem/assets/images/filename.jpg`
3. For _config.yml or theme settings, use `/assets/images/filename.jpg`
